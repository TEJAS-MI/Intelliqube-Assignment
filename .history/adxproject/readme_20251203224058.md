 ADX Calculator (Excel-Matched) – Django Web 
 
 A web-based ADX (Average Directional Index) calculator built using Python + Django, designed to exactly match Excel output using Wilder’s smoothing formula.

Users can upload an OHLC CSV, generate:
TR
+DM / -DM
TR14

DM14

DI values

DX

ADX (Wilder smoothing)

…and download the results or view them in a graph.
